Title: CP-ABE Cloud Security Application

Description: To provide the security without degrading the performance, we proposed an access control model featured with efficient key update function in data outsourcing environment.Our access control is based on Cipher text policy- Attribute based encryption(CP-ABE). In our project, a user's key is incorporated into the token which will be used to decrypt the Cipher text encrypted with CP-ABE policy. If we need the encrypted data to be decrypted then that particular key has to be provided and then the decrypted data for the user will be shared in the distributed system.Finally, data is highly secured as hybrid encryption on CP-ABE is used for encryption and decryption.
